### 2、请求头cookie和响应头set-cookie

#### 以教学研的单点登录为例子

- request header的Cookie

  当客户端（如浏览器）向服务器发送请求时，如果本地存储了与该域名匹配的 Cookie，浏览器会**自动**将 Cookie 附加到请求头的 `Cookie` 字段中

- response header的Set-Cookie

  服务器告诉客户端需要存储 Cookie
